# ScriptsForAutomation

Una raccolta di script Bash e Python nati per automatizzare task comuni su sistemi Windows e Linux (testati su Debian e Fedora).

## Script inclusi

### 1. 💿 PS2 ISO Creator (`crea_iso_ps2.sh`)
Uno script Bash per creare immagini ISO dai propri dischi originali per PlayStation 2.
* **Funzionalità**: Smonta automaticamente il disco, crea una copia 1:1 bit-a-bit e salva il file nella cartella desiderata.
* **Requisiti**: `dd`, `sudo`.
